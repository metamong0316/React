farr.find(callback[, thisArg])
        매개변수
        callback
        배열의 각 값에 대해 실행할 함수. 아래의 세 인자를 받습니다.
        element
        콜백함수에서 처리할 현재 요소.
        indexOptional
        콜백함수에서 처리할 현재 요소의 인덱스.
        arrayOptional
        find 함수를 호출한 배열.
        thisArg
        선택 항목. 콜백이 호출될 때 this로 사용할 객체.
        반환 값
        주어진 판별 함수를 만족하는 첫 번째 요소의 값. 그 외에는 undefined.
