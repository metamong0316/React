filter() 메서드는 조건 결과를 새로운 배열로 반환합니다.

        구문
        arr.filter(callback(element[, index[, array]])[, thisArg])

        arr.filter(function(현재값 => {
            return 조건
        })

        arr.filter(현재값 => return 조건)

        매개변수
        callback: 함수
        element : 현재 요소.
        index  : 인덱스

        반환 값
        조건에 맞는 새로운 배열.
